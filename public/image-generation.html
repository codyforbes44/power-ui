UX Workflow

From prompt to file in four steps.

No pipeline to configure. No API wrangling. Open the image generator, type a prompt, and download production-ready files.

1🔑

Add API Key

Paste your BFL, fal.ai, or Replicate API key in Settings. Keys are AES-256 encrypted immediately and never leave your device.

2🎨

Write Your Prompt

Describe what you want to generate. Use Claude's built-in prompt enhancement skill to automatically improve your prompt with cinematography and lighting detail.

3⚙️

Configure & Generate

Choose your provider, model, aspect ratio, steps, and guidance scale. Hit Generate and watch the image stream in real time.

4📥

Export Production Files

Download PNG, JPEG, or WebP at full resolution. Images are saved to your workspace history with full metadata — provider, model, prompt, seed, cost.
