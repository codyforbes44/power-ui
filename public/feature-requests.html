Community Roadmap

Shape what gets built next.

Vote on community ideas or submit your own. We read every request and use votes to prioritise the roadmap.

Community Ideas

● Planned UI / UX

Split-pane view for comparing two conversations side-by-side

When working on prompts, it's really useful to run two conversations with different system prompts and compare responses. A split view would save so much context-switching.

Suggested by @promptr 💬 14 comments
● Planned Integrations

Obsidian plugin to sync conversations and memories

An Obsidian plugin that automatically exports conversations as notes and syncs memory vault entries as a knowledge graph. Would make Async a core part of my PKM workflow.

Suggested by @neuron_net 💬 31 comments
● Reviewing AI / Models

Local model support via Ollama / LM Studio

Connect to locally-running models (Llama 3, Mistral, Phi-3) via the OpenAI-compatible API that Ollama and LM Studio expose. Zero cost, fully offline, maximum privacy.

Suggested by @offline_ai 💬 28 comments
● Reviewing Productivity

Keyboard shortcut cheat sheet & custom keybindings

A built-in shortcut reference (⌘/) and ability to remap any keyboard shortcut. Power users live in the keyboard — let us configure everything.

Suggested by @keyboardnerd 💬 9 comments
○ Open Sharing

One-click shareable conversation links with view-only mode

Generate a shareable permalink for a conversation that anyone can view (read-only). Optionally allow them to fork it into their own session. Great for sharing AI-assisted work.

Suggested by @share_it 💬 6 comments
○ Open Files

Drag-and-drop file upload with automatic context injection

Drag a PDF, image, or code file onto the chat window to have its contents automatically injected into the context. Support for Claude's vision API for images.

Suggested by @dropfiles 💬 11 comments
✓ Shipped in v1.4.0 Export

Export conversations as Markdown or JSON

Simple export to Markdown, JSON, or plain text so I can archive conversations in my own notes system or process them programmatically.

Suggested by @export_lover 💬 19 comments
