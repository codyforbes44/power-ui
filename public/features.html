05 · Security & Privacy

Local-first.
Encrypted by default.

🔐
AES-256-GCM encryption — API keys are encrypted at rest using a vault key derived from your password. Never stored in plaintext.
🛡️
PBKDF2-SHA256 — Passwords hashed with 310,000 iterations per NIST SP 800-132. Legacy accounts upgraded silently.
🔇
Zero telemetry — No analytics, no crash reports, no usage data sent anywhere.
Security Architecture
// Key derivation at login
const vaultKey = await deriveVaultKey(
password, salt, 310_000
);
// Encrypt before storage
const enc = await aesGcmEncrypt(apiKey, vaultKey);
// In-memory only, never persisted
sessionStorage.setItem('vaultKey', key);
