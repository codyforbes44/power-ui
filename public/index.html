Features

Everything claude.ai doesn't give you

Built for people who demand more from their AI tools — more control, more context, more power.

🧠

72-Skill Library

Skills are expert-level prompt injections that auto-activate based on your intent. Python debugging, legal drafting, data analysis — it knows when to apply them.

Python SQL Legal Docs +68 more
🔑

Bring Your Own Keys

Connect directly to provider APIs with your own keys. No margin, no middleman. Pay what the API actually costs — often 10× cheaper than subscriptions.

💾

Persistent Memory

Store and search facts across all your sessions. The app remembers your tech stack, preferences, project constraints — and injects them automatically.

🗂️

Workspaces

Organize conversations by project, client, or context. Each workspace has its own system prompt prefix, memory store, and session history.

🔒

Local-First & Encrypted

All data stays on your device. API keys are AES-GCM encrypted at rest, derived from your login password. No telemetry. No cloud sync unless you want it.

📊

Admin Dashboard

Full analytics, user management, API key governance, MCP resource management, and usage cost tracking — for teams and self-hosted deployments.

Quick Start

Up and running in 60 seconds

No build step. No account required. Just open the file and go.

1

Open the App

Download the release, open app/index.html in any modern browser — or run server.py for multi-device sync.

2

Add Your API Key

Paste your Anthropic, OpenAI, Google, or Groq API key in Settings. Keys are encrypted immediately — never stored in plaintext.

3

Start Chatting

The skills engine detects your intent and injects expert context automatically. Just talk — the platform handles the rest.

FAQ

Frequently asked questions

You create an API account directly with Anthropic, OpenAI, Google, or Groq, get your API key, and paste it into Settings. Requests go directly from your browser to the provider — no intermediary, no margin. You pay exactly what the provider charges.
Yes. All conversation history, settings, and memories are stored exclusively in your browser's localStorage (or your own server if you run server.py). API keys are AES-256-GCM encrypted using a key derived from your password — they are never stored in plaintext. We collect zero telemetry.
Skills are expert-level prompt injections from a library of 72 specialist modules (Python, SQL, legal writing, data analysis, etc.). The skills engine analyzes your message and automatically prepends the most relevant skill context to your API request. You can also manually browse and apply skills from the skills panel.
Yes. The built-in admin dashboard supports multi-user accounts with role-based access. For teams, we recommend the Team plan which includes shared workspaces, per-user analytics, and SSO. For self-hosted deployments, run server.py to enable multi-device sync and optionally put it behind your own authentication proxy.
The interface works offline — you can browse conversation history, edit sessions, and manage settings. Sending new messages requires an internet connection to reach the AI provider APIs. An offline banner appears automatically when connectivity is lost.
